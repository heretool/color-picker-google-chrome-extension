# Security Policy

## Supported Versions
This security policy applies to the `color-picker-google-chrome-extension` project maintained by HereTool.
Only the latest released version of this Chrome Extension receives security updates and vulnerability fixes.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability
If you discover any security vulnerability, flaw, or privacy risk within this color picker Chrome extension, please contact us **privately** via email instead of creating public GitHub Issues to prevent potential exploitation.

### Contact Information
- Official Email: support@heretool.com
- Official Website: https://heretool.com

### What to Include in Your Report
1. Full description of the security issue
2. Step-by-step reproduction instructions
3. Potential impact of the vulnerability
4. Suggested mitigation or fix (if available)

## Response Timeline
1. Acknowledge your vulnerability report within **3 business days** after receiving your email.
2. We will evaluate the severity of the issue and provide a scheduled fix timeline.
3. Once the patch is released, we will notify you via email and credit your contribution (if you wish to be acknowledged).

## Disclosure Rules
- We request a **90-day private disclosure window** from the date we confirm receipt of your valid vulnerability report.
- Public disclosure of any security flaw is prohibited before we release an official patched version.
- After we publish the fixed extension release, you are welcome to publicly share details of the resolved vulnerability.

## Additional Notes
This open-source Chrome color picker extension is hosted at https://github.com/heretool/color-picker-google-chrome-extension.
All security-related communications will be handled through support@heretool.com only.
For general feature requests or non-security bugs, please open a public GitHub Issue on the repository.
